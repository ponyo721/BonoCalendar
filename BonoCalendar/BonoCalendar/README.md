#  개발 정보 정리

## 화면 구성

0. 메인 화면
    - 원하는 날짜 빠른 이동 
    - 검색
    - 환경설정
1. 달력 (CalendarItem)
    - 타이틀
    - 대표 색상
    - 목표 (multiType: 마라톤, 가걔부 ....)
    - 탭뷰 아이콘 이미지 (only iphone)
    - 달력 내용 List
2. 달력 내용(CalendarContent)
    - 생성자 (2차 개발 시 캘린더 공유 기능 추가 예정)
    - 타이틀
    - 내용
    - 시작일
    - 종료일
    - 양력/음력 여부
